# Jak używać tego folderu (PL)

Cześć Bartek — to jest Twój gotowy pakiet aplikacyjny pod ofertę Booksy (AI-first
Software Engineer). Wszystko po angielsku, bo CV i portfolio wysyłasz po angielsku.

## Co masz
1. **CV_Bartosz_Gontarek_EN.html** — nowe CV w wersji tech/AI. Otwórz w przeglądarce →
   **Ctrl+P** → „Zapisz jako PDF" (Marginesy: brak, Grafika tła: włączona). Masz PDF.
2. **PORTFOLIO_CASE_STUDIES_EN.html** — szczegółowe opisy projektów (też → PDF).
   Wersje `.md` to to samo, ale do edycji, gdybyś chciał coś zmienić.
3. **projects/** — oczyszczony kod (bez żadnych Twoich kluczy API) + README do każdego
   projektu. To pokazujesz, jeśli ktoś poprosi o kod.
4. **certificates/** — lista 8 certyfikatów Anthropic + linki weryfikacyjne.

- Nowy tytuł: **AI-First Developer / Automation Builder** (zamiast „Sales Specialist").
