## ⭐ "Enlist the AI-assisted development tools you've experimented with. Which one is your favorite and why?" (wymagane — wklej)
> My favorite is **Claude Code**. I use it daily as my primary development environment — not as autocomplete, but as an agentic pair-engineer: I give it a goal, it reads my codebase, plans, edits across files, runs commands and verifies the result. I've shipped trading bots, automations and e-commerce work this way, and I extend it with **Model Context Protocol (MCP)** servers so the agent can call live tools (trading APIs, data sources, image generation) directly. I've also worked with **Cursor** (excellent in-editor inline flow) and **Gemini** (strong for multimodal/vision and image generation in my content pipelines). I hold Anthropic's *Claude Code in Action* and *MCP: Advanced Topics* certificates. What I like most about Claude Code is that it rewards good engineering thinking — clear specs, verification, breaking work into agentic steps — instead of hiding the process.
