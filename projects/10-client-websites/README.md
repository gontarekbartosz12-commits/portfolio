## Balustrady — full WordPress build pack (client)
A complete, ready-to-build website for a railings (balustrades) company. Lead-gen model
built around phone contact (phone = required field).

**What I produced (10-page site):**
- **Architecture & stack** — sitemap, mega-menu, WordPress child theme, custom post type
  "Realizacje" (projects) with category filtering + lightbox, Rank Math, SSL.
- **SEO for every page** — unique meta title/description, slugs, keywords, H1–H3 structure
  and schema.org markup across all 10 subpages.
- **Quote form** — spec with required phone, category pre-select, double opt-in email,
  GDPR + anti-spam.
- **Tracking** — GA4 + Facebook Pixel, conversion event, Consent Mode v2, Search Console.
- **Performance & images** — PageSpeed > 80 / < 3 s plan, image spec, naming, ALT texts.
- **Deployment checklist** — step-by-step from hosting to go-live.
- **Copy** — full Polish copy for the homepage + 6 product pages + Realisations, About,
  Contact/Quote (verified claims only; factual numbers left as `[TO FILL]` for the owner —
  no invented certifications).

**Status:** built and ready to deploy (a working prototype is included).
